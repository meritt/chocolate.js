SEO vs Контекст

Статьи, Проекты, пятница, 21 октября, 2011

Два примера из жизни для размышления.

Показать галерею

Пример 1

Наш клиент express-post.ru — продвигается в нашей компании с марта 2009 года по сей день.

Наш клиент inlingvo.ru — продвигается в нашей компании с ноября 2010 года по сей день.

У контекстной рекламы есть достоинства в виде прозрачности затрат, временного и географического таргетинга. Вы точно знаете сколько вы платите за каждого клиента по каждому запросу, все показы направлены на целевую аудиторию.

Наша компания разработала предложение, в котором присутствует прозрачность контекстной рекламы и отсутствует нестабильность алгоритмов поисковых систем — это КПМ (комплексный поисковый маркетинг). В рамках данного предложения мы продвигаем максимальное количество ваших запросов в поисковых системах и ведем контекстную рекламу. Причем вся работа анализируется для поиска максимально эффективных инструментов именно для вашей компании. Гарантией по данному предложению являются приведенные посетители на сайт из поисковых систем по целевым ключевым запросам. При сменах алгоритма мы увеличиваем нагрузку на контекстную рекламу. То есть наша задача - стабильно приводить на ваш сайт гарантированное количество посетителей.
